# Get start with swarm
[link](https://docs.docker.com/engine/swarm/swarm-tutorial/)

## Scripts

* 1-create-docker-machines.sh  
Create docker-machines using virtualbox provider. Remove previeus machine before re-execute this script

* 2-create-a-swarm.sh  
Create the swarm leader in swarm-host-1

* 3-add-swarm-nodes.sh  
Create swarm workers attached to swarm-host-1

* 4-deploy-service-to-swarm.sh  
Deploy a service to swarm, scale up and down
